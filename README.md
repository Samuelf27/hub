## 📌 Sobre

Uma página estilo "linktree para devs": agrega todos os projetos num só lugar, com cartões filtráveis por categoria (Biblioteca, CLI, Web App, API, Machine Learning...), cada um com link para a **demo ao vivo** e o **código**.

## ✨ Destaques

- 🗂️ Filtro por categoria com animação de entrada
- 🔗 Links diretos para demos e repositórios
- 📱 Totalmente responsivo
- ⚡ Estático e leve, sem dependências

## 🛠️ Tecnologias

HTML5 · CSS3 · JavaScript (vanilla)
